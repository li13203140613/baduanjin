# 音频文件说明

请将您的音频文件放置在此目录中：

## 站桩练习
- **文件名**: `meditation-music.mp3`
- **说明**: 站桩练习的背景音乐，每循环一次为15分钟

## 八段锦练习
八段锦练习使用程序化生成的呼吸音效，无需额外的音频文件。
- 吸气时会自动播放高频"嘶——"声
- 呼气时会自动播放低频"呼——"声
- 音效时长会根据您设置的呼吸时长自动调整
